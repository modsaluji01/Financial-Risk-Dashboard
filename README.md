📊 Financial Risk Analytics Dashboard (Power BI)
📌 Overview

This project presents an interactive Financial Risk Analytics Dashboard built using Power BI to analyze loan performance, default trends, and customer risk profiles. It helps in transforming raw financial data into meaningful insights for better decision-making.

🎯 Objectives
Analyze loan distribution and performance
Track default rates and risk trends
Perform customer segmentation based on demographics
Enable data-driven decision-making through visual insights
📊 Dashboard Features
📈 KPI Metrics: Total Loan Amount, Average Loan, Default Rate
📅 Trend Analysis: Year-over-Year (YOY) & Year-to-Date (YTD)
👥 Customer Segmentation: Credit Score, Income, Age, Marital Status
🔍 Interactive Filters: Dynamic slicers for deep analysis
📉 Risk Insights: Identification of high-risk customer groups
🛠️ Tools & Technologies
Power BI
Power Query (Data Cleaning & Transformation)
DAX (Calculated Measures & Metrics)
⚙️ Data Processing
Cleaned and transformed raw data using Power Query
Handled missing values and inconsistencies
Created custom calculations using DAX for deeper insights
💡 Key Insights
Identified high default rates in specific income and credit score segments
Observed trends in loan growth and repayment behavior
Highlighted customer profiles contributing to financial risk
🚀 Project Outcome

This dashboard enables stakeholders to quickly understand financial risk patterns and supports data-driven decision-making by providing clear, interactive, and actionable insights.
